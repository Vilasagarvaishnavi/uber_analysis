# Uber Data Analysis – Power BI Dashboard

An interactive and visually rich Power BI dashboard created to analyze Uber rides, revenue patterns, booking behavior, vehicle performance, customer experience, and location trends. This project showcases business insights through clean UI design, meaningful KPIs, and drill-down capabilities.

# Project Overview

This project focuses on analyzing Uber booking data to understand operational performance and customer behavior. The Power BI report provides a comprehensive view of how different vehicle types perform and how much revenue they generate.

The dashboard is designed with interactive cards, custom icons, navigation buttons, trend lines, revenue bars, and rating visuals.

# Objectives

* Understand how Uber rides are distributed across vehicle types
* Track completed, cancelled, and incomplete bookings
* Analyze monthly booking trends
* Identify top pickup and top drop locations
* Measure business performance using KPIs: revenue, distance, ratings
* Provide a detailed breakdown of each vehicle type with visuals
* Build a visually appealing and user-friendly Power BI interface

# Key Insights

## Ride Performance

* 93K+ Completed rides
* 57K+ Cancelled/Lost bookings
* 148K+ Total bookings

## Revenue Highlights

* Total Revenue: ₹5.18 Cr+
* Top contributors: Auto, Bike, Go Mini, Go Sedan
* Consistent performance across all quarters

## Distance & Patterns

* 2.51M km Total distance travelled
* 24.64 km Average distance per ride
* Clear monthly peaks showing seasonal demand changes

## Experience Metrics

* Avg Customer Rating: 4.40
* Avg Driver Rating: 4.23

## Hotspot Locations

* Top Pickup: Khandsa (600 rides)
* Top Drop: Ashram (592 rides)
